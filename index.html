Environment Variables

Add these to ~/.zshrc or ~/.bashrc

# Unified Endpoint Configuration

export OPENAI_BASE_URL="https://unified-endpoint-main.app.iu-it.org/openai/v1"

export OPENAI_API_KEY="YOUR_KEY"

export ANTHROPIC_BASE_URL="https://unified-endpoint-main.app.iu-it.org/anthropic"

export ANTHROPIC_AUTH_TOKEN="YOUR_KEY"

export GOOGLE_GEMINI_BASE_URL="https://unified-endpoint-main.app.iu-it.org/gemini/v1beta"

export GEMINI_API_KEY="YOUR_KEY"

$ source ~/.zshrc

Install OpenCode

Or use npm / homebrew:

$ brew install opencode
